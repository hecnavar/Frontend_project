HTML - Lenguaje de Marcado

La base estructural de cualquier sitio web.

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado utilizado para estructurar el contenido en la web. Permite definir elementos como encabezados, párrafos, imágenes y mucho más.

Ejemplo básico:

        
          <!DOCTYPE html>  
          <html>  
          
            <head>  
              <title>Mi página</title>  
            </head>
            
            <body>  
            
              <h1>Hola Mundo</h1>  
              <p>Este es un ejemplo básico de HTML.</p>  
              
            </body>
            
          </html>

Explicación de las etiquetas:

  • <!DOCTYPE html> - Indica que el documento es HTML5.
  • <html> - Etiqueta raíz que engloba todo el contenido.
  • <head> - Contiene metadatos e información del documento.
  • <title> - Define el título de la pestaña del navegador.
  • <body> - Contiene el contenido visible de la web.
  • <h1> - Encabezado de nivel 1 (el más grande).
  • <p> - Define un párrafo de texto.

Etiquetas esenciales en HTML

  • <h1> a <h6> - Encabezados
  • <p> - Párrafos
  • <a href="url"> - Enlaces
  • <img src="imagen.jpg"> - Imágenes
  • <table> - Tablas
  • <form> - Formularios
