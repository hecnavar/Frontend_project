¿Qué es Sass?

Sass es un preprocesador que extiende las capacidades de CSS, permitiéndote escribir código más limpio, organizado y reutilizable.

1. Variables

Las variables te permiten definir valores reutilizables, como colores o tamaños.

Fondo azul (definido como una variable en Sass) 

$primary-color: #3498db;
body {
  background-color: $primary-color;
}

2. Mixins

Los mixins son bloques de código que puedes reutilizar con diferentes parámetros.

Box con sombreado

@mixin shadow($color, $size) {
  box-shadow: 0 $size $size $color;
}

.box {
  @include shadow(rgba(0, 0, 0, 0.2), 10px);
}

3. Anidación

La anidación facilita la escritura de estilos jerárquicos, como elementos dentro de un `div`.

  • Item 2
  • Item 3

.nested-list {
  list-style: none;

  li {
    color: $primary-color;
    padding: 5px 0;
  }
}

4. Funciones

Sass incluye funciones como `darken` y `lighten` para manipular colores.

Texto original

Texto oscuro


$base-color: #2c3e50;

.text-original {
  color: $base-color;
}

.text-darker {
  color: darken($base-color, 10%);
}

Aprender Sass es un paso hacia CSS más organizado. 🌟
