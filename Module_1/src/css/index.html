CSS: Animaciones y Transiciones

Explora diferentes transiciones y animaciones usando tarjetas interactivas.

Rotación y Escalado

Pasa el cursor para ver el efecto.


.example-box {
    transition: transform 0.5s ease-in-out;
}
.example-box:hover {
    transform: rotate(45deg) scale(1.2);
}
Animación de Pulso

Animación continua de pulso.


@keyframes pulse {
    0%, 100% {
        transform: scale(1);
        opacity: 1;
    }
    50% {
        transform: scale(1.5);
        opacity: 0.6;
    }
}
.pulse-animation {
    animation: pulse 2s infinite;
}
