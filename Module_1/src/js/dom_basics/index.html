Introducción al DOM

El siguiente Ejemplo le pide nombre y edad al usuario y mediate el uso de DOM, no regresa un texto en la pagina con los datos introducidos

Del lado derecho se puede encontrar el código con comentarios de lo que se esta haciendo

Ejemplo Interactivo con DOM

Introduce un nombre y una edad para crear una persona:


        class Person {
            constructor(name, age) {
                this.name = name; // Almacena el nombre de la persona
                this.age = age;   // Almacena la edad de la persona
            }
        
            // Método que devuelve un mensaje de saludo
            greet() {
                return `Hola, soy ${this.name} y tengo ${this.age} años.`;
            }
        }
        
        // Agregamos un evento "click" al botón con id "createPerson"
        document.getElementById("createPerson").addEventListener("click", function() {
            // Obtener los valores ingresados y eliminar espacios en blanco
            const name = document.getElementById("name").value.trim();
            const age = document.getElementById("age").value.trim();
            
            // Referencia al div donde se mostrará el resultado
            const resultDiv = document.getElementById("result");
        
            // Validar que los campos no estén vacíos y que la edad sea un número válido
            if (name === "" || age === "" || isNaN(age) || age <= 0) {
                resultDiv.innerText = " Por favor, ingresa un nombre y una edad válida.";
                resultDiv.classList.add("error"); // Cambia el color del texto a rojo en caso de error
                return; // Detiene la ejecución si los datos no son válidos
            }
        
            // Crear un nuevo objeto de la clase "Person"
            const person = new Person(name, age);
        
            // Mostrar el saludo en el div de resultado
            resultDiv.innerText = person.greet();
            resultDiv.classList.remove("error"); // Restablecer el color del texto a negro
        });
