¿Qué es el DOM?

El DOM es una representación en forma de árbol de todos los elementos HTML de una página web. Cada elemento, atributo y texto de la página se convierte en un objeto que puede ser accesible y manipulable a través de JavaScript.

¿Cómo se manipula el DOM?

Para manipular el DOM con JavaScript, utilizamos métodos y propiedades que nos permiten modificar elementos, agregar nuevos, eliminar existentes o cambiar su contenido. Algunos de los métodos más comunes son:

  • document.getElementById(id): Permite acceder a un elemento de la página a través de su ID.
  • document.getElementsByClassName(class): Obtiene una lista de elementos con una clase específica.
  • document.createElement(tag): Crea un nuevo elemento HTML.
  • element.appendChild(node): Añade un nuevo nodo hijo a un elemento.
  • element.innerHTML: Permite modificar el contenido HTML de un elemento.

Ejemplo de Manipulación del DOM

A continuación, se muestra un ejemplo simple de cómo agregar un nuevo párrafo a la página mediante JavaScript: 


            // Crear un nuevo elemento de párrafo
            let nuevoParrafo = document.createElement("p");
    
            // Establecer el texto del párrafo
            nuevoParrafo.innerHTML = "Este es un párrafo agregado dinámicamente.";
    
            // Agregar el párrafo al cuerpo del documento
            document.body.appendChild(nuevoParrafo);

En este ejemplo, el código crea un nuevo elemento de tipo <p>, establece su contenido de texto con innerHTML y luego lo agrega al cuerpo del documento con appendChild.

Conclusión

La manipulación del DOM es esencial para crear aplicaciones web interactivas y dinámicas. A través de ella, podemos modificar el contenido, estructura y estilo de la página sin necesidad de recargarla, lo que mejora la experiencia del usuario.
